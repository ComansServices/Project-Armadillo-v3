# Armadillo v3 Scaffold (Option A)

TypeScript monorepo scaffold for:
- `apps/web` (Next.js)
- `apps/api` (NestJS)
- `apps/worker` (BullMQ)
- `packages/*` shared libs

## Quick Start
```bash
cd armadillo-v3
pnpm install
pnpm dev
```

## Services
- Web: http://localhost:3000
- API: http://localhost:4000

## Notes
This is an initial scaffold and planning baseline, not feature-complete.
