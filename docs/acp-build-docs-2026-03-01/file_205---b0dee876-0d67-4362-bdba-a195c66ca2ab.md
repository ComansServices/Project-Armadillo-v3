## Prisma Models (7 EXISTING Tables)

All models in `apps/api/prisma/schema.prisma`.

### Scan
Full scan lifecycle. Fields: id, projectId, status (queued/running/completed/failed), targets (JSON), config (JSON), startedAt, completedAt, timestamps.

### ScanEvent
Event log per scan. Fields: id, scanId, stage, status, message, data (JSON), timestamps.

### ScanSchedule
Cron-based recurring scans. Fields: id, projectId, cronExpression, targets, paused (boolean), pauseReason, lastRunAt, nextRunAt, timestamps.

### XmlImport
Quality-controlled XML import records. Fields: id, projectId, source, qualityMode (strict/lenient), status, alertTriggered (boolean), qualityStats (JSON), rejectArtifactPath, timestamps.

### ImportSourcePolicy
Trusted import source governance. Fields: id, name, qualityMode, description, timestamps.

### Asset
Network assets with identity dedup. Fields: id, importId, projectId, ip, hostname, os, ports (JSON), services (JSON), identityKey (unique dedup key), deltaSinceLast (JSON change tracking), seenCount, firstSeenAt, lastSeenAt, timestamps.

### AssetVulnerability
CVE/finding linked to asset. Fields: id, assetId, projectId, cveId, severity, description, hasExploit (boolean), blastRadius (integer), exploitRefs (String[]), assignedTo, dueDate, remediationStatus (open/in_progress/on_hold/resolved), statusReason, timestamps.

### NOT YET BUILT
