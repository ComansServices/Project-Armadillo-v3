## Existing Models (7 tables — in production)

### Scan
```prisma
model Scan {
  id            String      @id @default(cuid())
  projectId     String
  status        String      @default("queued")  // queued|running|completed|failed
  targets       Json                            // [{type, value}]
  config        Json?                           // Pipeline config
  startedAt     DateTime?
  completedAt   DateTime?
  events        ScanEvent[]
  createdAt     DateTime    @default(now())
  updatedAt     DateTime    @updatedAt
  @@map("scans")
  @@index([projectId])
  @@index([status])
}
```

## Index Strategy

Every field used in WHERE clauses MUST have a `@@index`. Key patterns:
- `@@index([projectId])` on ALL tenant-scoped tables
- `@@index([cveId])` on vulnerability and intelligence tables
- `@@index([createdAt])` on time-series tables
- `@@index([status])` on state-machine tables
- `@@unique([userId, projectId])` on membership tables
- Composite indexes for common query patterns
